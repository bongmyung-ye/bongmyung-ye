# Profile Roadmap

이 문서는 GitHub 프로필과 공개 프로젝트를 개선하기 위한 작업 계획을 정리합니다.

## Short Term

- 프로필 README 레이아웃 정리
- 대표 프로젝트 설명 보강
- 사용 기술 스택 정리
- 공개 저장소 고정 설정

## Project Showcase

- Discord Music Bot
- Discord Statistics Bot
- Moderation / Role Bot
- Portfolio Website

## Improvements

- README 스크린샷 추가
- 설치 및 실행 방법 문서화
- 환경 변수 예시 파일 정리
- 프로젝트별 기능 목록 작성

## Goal

프로필을 단순한 소개 페이지가 아니라, 실제로 만들고 운영한 프로젝트를 보여주는 포트폴리오로 개선합니다.
